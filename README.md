# Express-Daemon-BoilerPlate
Express/Mongo Daemon BoilerPlate

### Description
A BoilerPlate for Creating an Express Server that can start/stop/restart/status from the commandline   

##### Features
- Runs as a Daemon
- Connects to MongoDB
- Drop in routes/middlewares
- Expandable With Drop In Components/Plugins
- Built in such a way that each component can communicate with eachother
- Commandline Tool to Interface with the Server
